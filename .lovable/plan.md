

# TOTO Finance Investor Dashboard

A clean, minimal investor dashboard with gold accents, featuring deal discovery, KYC verification, and investment tracking.

---

## 1. Authentication Pages

### Sign In Page
- Clean centered card with TOTO Finance branding
- Email & password fields with minimal styling
- **Sign in with Google** button with Google logo
- **Sign in with X** button with X logo
- Link to sign up for new users
- Forgot password link

### Sign Up Page
- Name, email, password, confirm password fields
- Social signup options (Google, X)
- Terms & privacy checkbox
- Link to sign in for existing users

---

## 2. Main Dashboard Layout

### Sidebar Navigation (Collapsible)
- TOTO Finance logo at top
- **Dashboard** - Deal discovery grid
- **My Investments** - Portfolio tracking
- **Profile/Settings** - Account & KYC status
- User avatar and logout at bottom

### Dashboard Content (6-8 Deal Cards)
Each card displays:
- Deal title & category tag
- Brief description
- Target amount with progress bar
- Minimum investment
- Expected returns/APY
- Risk level indicator (Low/Medium/High)
- Funding deadline
- "View Details" button with gold accent

---

## 3. Deal Detail Page

### Deal Information Section
- Large header image/banner
- Deal title with status badge
- Full description & terms
- Investment metrics (target, raised, investors, time left)
- Risk & returns breakdown
- Documents available for download

### Investment CTA Section
- Prominent "Invest Now" button (gold accent)
- Minimum/maximum investment amounts shown
- Key highlights summary

---

## 4. KYC Flow (Before First Investment)

### Multi-Step KYC Modal
**Step 1 - Personal Information**
- Full legal name, date of birth
- Phone number, address

**Step 2 - Identity Verification**
- ID type selection (passport, driver's license, national ID)
- Upload front & back of document
- Selfie upload for verification

**Step 3 - Confirmation**
- Review submitted information
- Verification pending status

---

## 5. Payment Options Modal

### After KYC Completion
Two payment method cards:

**Bank Transfer Option**
- Bank account details displayed
- Reference number for transfer
- Instructions for completing transfer

**Crypto Payment Option**
- Wallet address display with copy button
- QR code for wallet
- Supported cryptocurrencies listed
- Amount conversion display

---

## 6. My Investments Section

### Portfolio Overview
- Total invested amount
- Current portfolio value
- Overall returns percentage
- Active/Pending/Completed summary

### Investment Cards
Each investment shows:
- Deal name and thumbnail
- Amount invested & date
- Current value & returns
- Status badge (Pending, Active, Completed)
- Performance chart/indicator
- View documents button

### Documents Tab
- Downloadable investment confirmations
- Ownership certificates
- Transaction receipts

---

## Design Implementation

- **Colors**: White backgrounds, near-black text, gold (#C4A052) accents
- **Typography**: Inter font, light weight headings (300), regular body
- **Shadows**: Soft card shadows, gold glow on hover states
- **Spacing**: Generous whitespace, clean grid layouts
- **Animations**: Subtle fade-ins, smooth hover transitions
